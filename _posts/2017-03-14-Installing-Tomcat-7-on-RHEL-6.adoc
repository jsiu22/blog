`1.` First make sure Java 7 or higher is installed.
....
java -version
....

`2.` Create tomcat user and group.
....
sudo groupadd tomcat
sudo mkdir /opt/tomcat
sudo useradd -s /sbin/nologin -g tomcat -d /opt/tomcat tomcat
....
